# Keybinds Hotkeys
On the keyboard, the `Meta` key acts as the Windows key, and the Alt key serves as the `Super` key. This configuration on your setup is the same across Windows, SteamOS, and macOS.

## Rebinding
- `Caps Lock` to `Meta (Left)`
- `Meta (Left)` to `` ` ``

## MacOS Style Copy/Paste/Cut/Select
- `Super (Left)` + `A`: Select
- `Super (Left)` + `C`: Copy
- `Super (Left)` + `V`: Paste
- `Super (Left)` + `X`: Cut

## Vim Style Movement
- `Super (Right)` + `H`: Left
- `Super (Right)` + `J`: Down
- `Super (Right)` + `K`: Up
- `Super (Right)` + `L`: Right

## Vim Style Escape
- `Meta (Left)` + `[`: Escape
