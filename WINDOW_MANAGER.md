# Window Manager Hotkeys
On the keyboard, the `Meta` key acts as the Windows key, and the Alt key serves as the `Super` key. This configuration on your setup is the same across Windows, SteamOS, and macOS.

## Focus Windows
- `Super` + `H`: Focus the window to the **west**
- `Super` + `J`: Focus the window to the **south**
- `Super` + `K`: Focus the window to the **north**
- `Super` + `L`: Focus the window to the **east**

## Move Windows
- `Shift` + `Super` + `H`: Swap the window to the **west**
- `Shift` + `Super` + `J`: Swap the window to the **south**
- `Shift` + `Super` + `K`: Swap the window to the **north**
- `Shift` + `Super` + `L`: Swap the window to the **east**

- `Super` + `6`: Focus workspace 6
- `Super` + `7`: Focus workspace 7
- `Super` + `8`: Focus workspace 8
- `Super` + `9`: Focus workspace 9

## Move Workspace
- `Shift` + `Super` + `1`: Move window to workspace 1 and focus it
- `Shift` + `Super` + `2`: Move window to workspace 2 and focus it
- `Shift` + `Super` + `3`: Move window to workspace 3 and focus it
- `Shift` + `Super` + `4`: Move window to workspace 4 and focus it
- `Shift` + `Super` + `5`: Move window to workspace 5 and focus it
- `Shift` + `Super` + `6`: Move window to workspace 6 and focus it
- `Shift` + `Super` + `7`: Move window to workspace 7 and focus it
- `Shift` + `Super` + `8`: Move window to workspace 8 and focus it
- `Shift` + `Super` + `9`: Move window to workspace 9 and focus it
- `Shift` + `Super` + `0`: Move window to workspace 10 and focus it

## Open Applications
- `Meta` + `1`: Open **Browser** in workspace 1
- `Meta` + `2`: Open **Terminal** in workspace 2
- `Meta` + `3`: Open **Editor** in workspace 3
- `Meta` + `4`: Open **Endpoint** in workspace 4
- `Meta` + `5`: Open **Notes** in workspace 5
- `Meta` + `6`: Open **Stream** in workspace 6
- `Meta` + `7`: Open **Chat** in workspace 7
- `Meta` + `8`: Open **Vault** in workspace 8
- `Meta` + `9`: Open **Music** in workspace 9
- `Meta` + `0`: Open **Message** in workspace 0

## Resize Windows
- `Super` + `U`: Resize window **left**
- `Super` + `P`: Resize window **right**
- `Super` + `I`: Resize window **bottom**
- `Super` + `O`: Resize window **top**

## Layout Orientation
- `Super` + `-`: Insert window or Change orientation to **east**
- `Super` + `\`: Insert window or Change orientation to **south**

## Window Controls
- `Super` + `Space`: Enable **float**
- `Shift` + `Super` + `Space`: Disable **float**
- `Shift` + `Super` + `Q`: Close window

## Reload Config
- `Shift` + `Super` + `R`: Reload Config
