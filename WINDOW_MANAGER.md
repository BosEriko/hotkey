# Window Manager Hotkeys

On the keyboard, the `Meta` key acts as the Windows key, and the Alt key serves as the `Super` key. This configuration on your setup is the same across Windows, SteamOS, and macOS.

## Focus Windows
- `Super` + `H`: Focus the window to the **west**
- `Super` + `J`: Focus the window to the **south**
- `Super` + `K`: Focus the window to the **north**
- `Super` + `L`: Focus the window to the **east**

## Move Windows
- `Shift` + `Super` + `H`: Swap the window to the **west**
- `Shift` + `Super` + `J`: Swap the window to the **south**
- `Shift` + `Super` + `K`: Swap the window to the **north**
- `Shift` + `Super` + `L`: Swap the window to the **east**

- `Shift` + `Super` + `6`: Move window to workspace 6 and focus it
- `Shift` + `Super` + `7`: Move window to workspace 7 and focus it
- `Shift` + `Super` + `8`: Move window to workspace 8 and focus it
- `Shift` + `Super` + `9`: Move window to workspace 9 and focus it
- `Shift` + `Super` + `0`: Move window to workspace 10 and focus it

## Open Applications
- `Meta` + `1`: Open **Zen** in workspace 1
- `Meta` + `2`: Open **Ghostty** in workspace 2
- `Meta` + `3`: Open **Zed** in workspace 3
- `Meta` + `4`: Open **Postman** in workspace 4
- `Meta` + `5`: Open **Obsidian** in workspace 5
- `Meta` + `9`: Open **Spotify** in workspace 9
- `Meta` + `0`: Open **Discord** in workspace 0

## Resize Windows
- `Super` + `U`: Resize window **left** by 20
- `Super` + `P`: Resize window **right** by 20
- `Super` + `I`: Resize window **bottom** by 20
- `Super` + `O`: Resize window **top** by 20

## Next Layout Orientation
- `Super` + `-`: Insert window to **east**
- `Super` + `\`: Insert window to **south**

## Window Controls
- `Super` + `Space`: Toggle window **float**
- `Shift` + `Super` + Space`: Toggle window **float**
- `Shift` + `Super` + Q`: Close window

## Reload Yabai Config
- `Shift` + `Super` + `R`: Reload Yabai and related services
